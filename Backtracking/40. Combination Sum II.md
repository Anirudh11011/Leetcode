## 40. Combination Sum II

🔗 https://leetcode.com/problems/combination-sum-ii/

---

## 🧩 Problem

Given a collection of candidate numbers `candidates` and a target number `target`, find all **unique combinations** in `candidates` where the candidate numbers sum to `target`.

Each number in `candidates` may only be used **once** in the combination.

**Note:** The solution set must not contain duplicate combinations.

---

## 📌 Examples

### Example 1:

**Input:**

```python
candidates = [10,1,2,7,6,1,5], target = 8
```

**Output:**

```python
[
    [1,1,6],
    [1,2,5],
    [1,7],
    [2,6]
]
```

---

### Example 2:

**Input:**

```python
candidates = [2,5,2,1,2], target = 5
```

**Output:**

```python
[
    [1,2,2],
    [5]
]
```

---

## 🔒 Constraints

* `1 <= candidates.length <= 100`
* `1 <= candidates[i] <= 50`
* `1 <= target <= 30`

---

## 💡 Approach (Backtracking + Sorting)

* First, sort the `candidates` array.
* Use **backtracking** to explore all possible combinations.
* At each step:

  * Pick the current number and move to the next index.
  * Since each number can only be used once, recurse with `i + 1`.
* To avoid duplicate combinations:

  * Skip repeated numbers at the same recursion level using:

    ```python
    if i > index and candidates[i] == candidates[i - 1]:
        continue
    ```
* Stop when:

  * Current sum equals target → add the combination to the result.
  * Current sum exceeds target → stop exploring that path.

---

## ⏱ Complexity

* **Time Complexity:** `O(2^n)` in the worst case
* **Space Complexity:** `O(n)` for recursion stack and path storage

---

## 🧠 Solution (Python)

```python
class Solution:
    def combinationSum2(self, candidates: List[int], target: int) -> List[List[int]]:
        def back(candidates, target, path, csum, index):
            if csum == target:
                res.append(path[:])
                return
            elif csum > target:
                return

            for i in range(index, len(candidates)):
                if i > index and candidates[i] == candidates[i - 1]:
                    continue

                path.append(candidates[i])
                csum += candidates[i]
                back(candidates, target, path, csum, i + 1)
                csum -= candidates[i]
                path.pop()

        res = []
        candidates = sorted(candidates)
        back(candidates, target, [], 0, 0)
        return res
```
