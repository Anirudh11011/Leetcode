## 784. Letter Case Permutation

🔗 https://leetcode.com/problems/letter-case-permutation/

---

## 🧩 Problem

Given a string `s`, you can transform every letter individually to be lowercase or uppercase to create another string.

Return a list of all possible strings we could create. Return the output in any order.

---

## 📌 Examples

### Example 1:

**Input:**

```python
s = "a1b2"
```

**Output:**

```python
["a1b2","a1B2","A1b2","A1B2"]
```

---

### Example 2:

**Input:**

```python
s = "3z4"
```

**Output:**

```python
["3z4","3Z4"]
```

---

## 🔒 Constraints

* `1 <= s.length <= 12`
* `s` consists of lowercase English letters, uppercase English letters, and digits.

---

## 💡 Approach (Backtracking)

* Use **backtracking** to explore all possible case transformations.
* At each index:

  * If the character is a **letter**:

    * Recurse with lowercase.
    * Recurse with uppercase.
  * If the character is a **digit**:

    * Just continue recursion with the same character.
* Stop when:

  * The current path length equals `len(s)` → add to result.

---

## ⏱ Complexity

* **Time Complexity:** `O(2^n)` (each letter can have 2 choices)
* **Space Complexity:** `O(n)` for recursion stack

---

## 🧠 Solution (Python)

```python
class Solution:
    def letterCasePermutation(self, s: str) -> List[str]:
        def back(index, path):
            if index == len(s):
                res.append(path)
                return

            if s[index].isalpha():
                back(index + 1, path + s[index].lower())
                back(index + 1, path + s[index].upper())
            else:
                back(index + 1, path + s[index])

        res = []
        back(0, "")
        return res
```
