## 422. Valid Word Square

🔗 https://leetcode.com/problems/valid-word-square/

---

## 🧩 Problem

Given an array of strings `words`, return `true` if it forms a **valid word square**.

A sequence of strings forms a valid word square if the **k-th row and k-th column read the same string**, where:

```
0 <= k < max(numRows, numColumns)
```

---

## 📌 Examples

### Example 1:

**Input:**

```
words = ["abcd","bnrt","crmy","dtye"]
```

**Output:**

```
true
```

**Explanation:**

* Row 1 = Column 1 → `"abcd"`
* Row 2 = Column 2 → `"bnrt"`
* Row 3 = Column 3 → `"crmy"`
* Row 4 = Column 4 → `"dtye"`

---

### Example 2:

**Input:**

```
words = ["abcd","bnrt","crm","dt"]
```

**Output:**

```
true
```

**Explanation:**
Even with uneven lengths, rows and columns still match:

* Row 3 = Column 3 → `"crm"`
* Row 4 = Column 4 → `"dt"`

---

### Example 3:

**Input:**

```
words = ["ball","area","read","lady"]
```

**Output:**

```
false
```

**Explanation:**

* Row 3 = `"read"`
* Column 3 = `"lead"`

Mismatch ❌ → Not a valid word square.

---

## 🔒 Constraints

* `1 <= words.length <= 500`
* `1 <= words[i].length <= 500`
* `words[i]` contains only lowercase English letters

---

## 💡 Approach

* Iterate through each row index `i`
* Build the corresponding column string:

  * For each word `j`, if index `i` exists → append `words[j][i]`
* Compare:

  * Row string `words[i]`
  * Constructed column string
* If any mismatch → return `False`

---

## ⏱ Complexity

* **Time Complexity:** `O(n * m)`

  * `n` = number of words
  * `m` = max length of a word
* **Space Complexity:** `O(m)` (temporary string)

---

## 🧠 Solution (Python)

```python
class Solution:
    def validWordSquare(self, words: List[str]) -> bool:
        
        for i in range(len(words)):
            temp = ''
            
            for j in range(len(words)):
                if i < len(words[j]):
                    temp += words[j][i]

            if temp != words[i]:
                return False

        return True
```
