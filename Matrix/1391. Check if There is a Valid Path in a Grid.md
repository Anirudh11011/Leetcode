## 1391. Check if There is a Valid Path in a Grid

🔗 https://leetcode.com/problems/check-if-there-is-a-valid-path-in-a-grid/

---

## 🧩 Problem

You are given an `m x n` grid. Each cell of `grid` represents a street. The street of `grid[i][j]` can be:

* `1` → connects **left ↔ right**
* `2` → connects **up ↕ down**
* `3` → connects **left ↔ down**
* `4` → connects **right ↔ down**
* `5` → connects **left ↔ up**
* `6` → connects **right ↔ up**

You start at the **top-left cell (0, 0)**.

A valid path is one that:

* Starts at `(0, 0)`
* Ends at `(m - 1, n - 1)`
* Follows valid street connections between adjacent cells

🚫 You **cannot modify** the streets.

Return `true` if such a path exists, otherwise return `false`.

---

## 📌 Examples

### Example 1:

**Input:**

```
grid = [[2,4,3],[6,5,2]]
```

**Output:**

```
true
```

**Explanation:**
You can follow connected streets from `(0,0)` to `(m-1,n-1)`.

---

### Example 2:

**Input:**

```
grid = [[1,2,1],[1,2,1]]
```

**Output:**

```
false
```

**Explanation:**
You get stuck at `(0,0)` since no valid connection exists.

---

### Example 3:

**Input:**

```
grid = [[1,1,2]]
```

**Output:**

```
false
```

**Explanation:**
You get stuck at `(0,1)` and cannot proceed.

---

## 🔒 Constraints

* `m == grid.length`
* `n == grid[i].length`
* `1 <= m, n <= 300`
* `1 <= grid[i][j] <= 6`

---

## 💡 Approach (BFS / Graph Traversal)

* Treat each cell as a node in a graph.
* Each street type defines **valid movement directions**.
* Use **BFS (Breadth-First Search)** starting from `(0,0)`:

  * Explore all reachable cells.
  * Move only if:

    * The neighbor is within bounds
    * The neighbor connects back to the current cell
* Use a `visited` set to avoid cycles.

---

## ⏱ Complexity

* **Time Complexity:** `O(m * n)`
* **Space Complexity:** `O(m * n)`

---

## 🧠 Solution (Python)

```python
class Solution:
    def hasValidPath(self, grid: List[List[int]]) -> bool:
        
        if len(grid) == 1 and len(grid[0]) == 1:
            return True

        directions = {
            1: [[0, -1], [0, 1]],
            2: [[-1, 0], [1, 0]],
            3: [[0, -1], [1, 0]],
            4: [[0, 1], [1, 0]],
            5: [[0, -1], [-1, 0]],
            6: [[0, 1], [-1, 0]]
        }

        queue = [[0, 0]]
        visited = set()
        visited.add((0, 0))

        while queue:
            r, c = queue.pop(0)
            street_type = grid[r][c]

            for dr, dc in directions[street_type]:
                nr, nc = r + dr, c + dc

                if (0 <= nr < len(grid) and 
                    0 <= nc < len(grid[0]) and 
                    (nr, nc) not in visited):

                    for back_r, back_c in directions[grid[nr][nc]]:
                        if nr + back_r == r and nc + back_c == c:
                            queue.append([nr, nc])
                            visited.add((nr, nc))

                            if nr == len(grid) - 1 and nc == len(grid[0]) - 1:
                                return True

        return False
```
